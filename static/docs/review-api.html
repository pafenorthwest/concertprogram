Concert Program Review APIs (0.5.0)

Download OpenAPI specification:

Endpoints used by reviewers to manage musical piece review metadata and queue assignment. Accepts either the bearer token (Authorization: Bearer ${auth_code}) or the pafe_auth session cookie used by the app.

list review queue items for the current reviewer

Returns musical pieces assigned to the authenticated reviewer for a specific division. Accepts either the bearer token (Authorization: Bearer ${auth_code}) or the pafe_auth session cookie. The queue requires a valid session cookie to identify the reviewer and determine their role (Admin, MusicEditor, or DivisionChair).

Authorizations:
PafeAuthCookieBearerAuth

division
required
string (DivisionTag)
Enum: "Violin Viola" "Cello Bass" "Piano" "Woodwinds"

replace the categories assigned to a musical piece

Replaces all category assignments for the musical piece with the supplied set. Not Appropriate takes precedence and will replace other categories if provided. Accepts either the bearer token (Authorization: Bearer ${auth_code}) or the pafe_auth session cookie.
